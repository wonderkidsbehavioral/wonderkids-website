WONDERKIDS BEHAVIORAL SERVICES — NETLIFY UPLOAD

1. Upload this entire folder (or this ZIP) to Netlify Deploys.
2. index.html must remain at the top level next to the assets folder.
3. All approved photos are stored as separate files inside assets/.
4. Image URLs include a cache-busting version so Netlify and browsers load the corrected photos.
5. After deployment, use Netlify: Deploys > Trigger deploy > Clear cache and deploy site.
6. Test the Netlify URL in a private browser window before connecting the custom domain.

Contact form submissions appear in Netlify Forms. Do not request sensitive medical details through a standard form unless your intake workflow is HIPAA-capable.
